# Elegant Birthday Animation in Java 🎂✨

This is a visually rich birthday animation built using Java Swing and Graphics2D.

## Features
- Animated "Happy Birthday" text with glowing effect
- Moving colorful balloons 🎈
- Gradient background
- Custom-drawn birthday cake 🎂
- Smooth animation using Timer

## Technologies Used
- Java
- Swing (JFrame, JPanel)
- Graphics2D
- Timer (animation)

## How to Run

1. Compile:
   javac ElegantBirthday.java

2. Run:
   java ElegantBirthday

## Output
- Animated birthday greeting with moving elements and glowing effects
